## ⚙️ Getting Started

To run the project locally:

```bash
# Clone the repo
git clone https://github.com/saadnadeem07/portfolio.git
cd portfolio

# Install dependencies
npm install

# Start development server
npm start

# To build for production:
npm run build

# To deploy to GitHub Pages:
npm run deploy
```
